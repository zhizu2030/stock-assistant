## 🔧 高级配置

### 添加更多股票

编辑 `src/utils/realData.ts` 文件，在 `yahooStockMapping` 和 `baseStocks` 中添加你想要的股票：

```typescript
const yahooStockMapping: Record<string, { code: string; name: string }> = {
  // 已有股票...
  '你的股票代码': { code: 'Yahoo代码', name: '股票名称' },
};

export const baseStocks: Stock[] = [
  // 已有股票...
  { code: '你的股票代码', name: '股票名称', price: 0, change: 0, changePercent: 0, volume: 0, high: 0, low: 0, open: 0 },
];
```

### Yahoo 股票代码格式

- 美股：直接使用股票代码（如 AAPL, MSFT）
- A股（上海）：代码.SS（如 600519.SS）
- A股（深圳）：代码.SZ（如 000001.SZ）
- 港股：代码.HK（如 0700.HK）

### 切换到更专业的数据源

如果需要更实时、更专业的数据，可以考虑：

#### 方案1：iTick API（推荐）
- 官网：https://itick.org
- 免费额度：基本行情无限调用
- 覆盖：A股、港股、美股、全球市场
- 延迟：<50ms
- 支持：REST 和 WebSocket

#### 方案2：Tushare Pro
- 官网：https://tushare.pro
- 免费额度：每日 500 次调用
- 覆盖：A股为主
- 数据质量：高

#### 方案3：Alpha Vantage
- 官网：https://www.alphavantage.co
- 免费额度：每日 25 次调用
- 覆盖：全球市场
- 特点：技术指标丰富

## ❓ 常见问题

**Q: 为什么有些股票没有数据？**
A: 可能是股票代码在 Yahoo Finance 中的格式不同，请检查 `realData.ts` 中的代码映射。

**Q: 如何添加更多股票？**
A: 编辑 `src/utils/realData.ts` 文件，添加到映射表和基础股票列表中。

**Q: 可以接入实时Level2数据吗？**
A: 免费数据源通常不提供Level2数据，需要使用付费API。

**Q: AI助手是真实的吗？**
A: 当前版本是模拟的回复，后续可以接入 OpenAI、Claude 等真实AI服务。
